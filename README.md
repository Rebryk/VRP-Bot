# Voice recognition bot

Send an audio message and get a recognized text! <br>
[VK community](https://vk.com/vrp_bot)

### VK config structure
```
{
  "server_host": <asr server>,
  "app_host": <host>,
  "app_port": <port>,
  "access_token": <access token>,
  "confirmation_key": <vk confirmation key>
}
```
